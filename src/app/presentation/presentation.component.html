La Compagnie Libre

Noe, créée 3 jours après la sortie de Final Fantasy XIV: A Realm Reborn est une compagnie axée Pve.
Nous sommes également présents en PvP par moment, histoire de se changer les idées, mais nos objectifs sont avant tout PvE.
Notre but est de progresser dans le contenu HL tout en s'éclatant ensemble, parce qu'après tout,
on joue pour ça !

Nous avons deux roster à l'heure actuelle, avec un rythme qui leur est propre (1 run par semaine avec possibilité de run le week end si tout le monde est présent).

Si vous souhaitez nous rejoindre pour accompagner l'un de nos roster ou même en créer un, n'hésitez pas à nous contacter IG (liste des membres ci-contre) ou bien via discord (voir onglet "Nous Rejoindre")
